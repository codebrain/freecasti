## Display (`nibble_hilo` @ 146–147)

Front-panel UI focus code (not a sound parameter). **Browse** (`92 = 02`): stable menu-row highlight — indices 1–17 follow `menu_index + 28`; reverb time (index 0) → `0x2E` (46); idle → 28. **Edit** (`92 = 00`): value-focus code in a parameter-specific band that advances as the shown value changes — not a fixed per-parameter anchor. See [bytes/display.md](bytes/display.md).
